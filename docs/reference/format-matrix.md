# Format Matrix

| Format | Open | Edit | Preview | Export | Toolbar | Search Support |
|--------|------|------|---------|--------|---------|----------------|
| Markdown (`.md`, `.markdown`) | Yes | Yes | Yes | Yes | Markdown | Enabled by default |
| JSON (`.json`) | Yes | Yes | No | No | JSON format | Enabled by default |
| YAML (`.yaml`, `.yml`) | Yes | Yes | No | No | YAML format | Enabled by default |
| Plain text (`.txt`) | Yes | Yes | No | No | None | Enabled by default |
| JSONL (`.jsonl`) | Yes | Viewer | No | No | None | Disabled by default |
| PDF (`.pdf`) | Yes | No | Yes | No | None | Disabled by default |
| DOCX (`.docx`) | Yes | No | Yes | No | None | Disabled by default |

## Notes

- PDF preview uses Qt WebEngine's PDF viewer.
- DOCX preview uses pandoc conversion to HTML.
- Block parsing and sync are markdown-only.
