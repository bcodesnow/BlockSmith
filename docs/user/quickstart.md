# Quickstart

## 1. Add Project Paths

1. Open Settings (`Ctrl+,`).
2. In Projects, add one or more search paths.
3. Keep default trigger files or customize them.
4. Run scan (`Ctrl+Shift+S` or `F5`).

## 2. Open and Edit Files

- Use the project tree on the left.
- Use quick switcher (`Ctrl+P`) for fast file jumping.
- Open multiple files: each file opens in a tab.

## 3. Use Markdown Workflow

- Toggle editor modes with `Ctrl+E` (Edit, Split, Preview).
- Use toolbar or shortcuts for formatting.
- Use find/replace (`Ctrl+F`, `Ctrl+H`).

## 4. Reusable Blocks

1. Select text in editor.
2. Right-click and choose "Add as Block".
3. Push/pull updates from the right pane.
4. Resolve divergences with diff popup when needed.

## 5. Export

- Press `Ctrl+Shift+E`.
- Choose PDF, HTML, or DOCX.
- Pick output path and export.

## 6. Optional Claude Code Integration

- Enable "Include Claude Code folder" in Settings > Integrations.
- Open `.jsonl` transcripts directly in the tree.
