# Keyboard Shortcuts

## Core

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Save current document |
| `Ctrl+R` | Reload current document |
| `Ctrl+Shift+S` / `F5` | Scan projects |
| `Ctrl+,` | Open settings |
| `Ctrl+Q` | Quit |

## Tabs and Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+P` | Quick switcher |
| `Ctrl+W` | Close active tab |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Ctrl+Shift+T` | Reopen closed tab |
| `Alt+Left` | Navigate back |
| `Alt+Right` | Navigate forward |
| Mouse back/forward buttons | Navigate back/forward |

## Editor

| Shortcut | Action |
|----------|--------|
| `Ctrl+E` | Cycle edit/split/preview (markdown) |
| `Ctrl+F` | Find |
| `Ctrl+H` | Replace |
| `Ctrl+D` | Duplicate line |
| `Ctrl+B` | Bold |
| `Ctrl+I` | Italic |
| `Ctrl+Shift+K` | Inline code |
| `Tab` / `Shift+Tab` | Indent / outdent |
| `Enter` | Continue list item (markdown contexts) |
| `Shift+Enter` | Find previous match |

## Search and Export

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+F` | Global search |
| `Ctrl+Shift+E` | Export dialog |

## Zoom

| Shortcut | Action |
|----------|--------|
| `Ctrl+=` / `Ctrl++` | Zoom in |
| `Ctrl+-` | Zoom out |
| `Ctrl+0` | Reset zoom |
| `Ctrl+MouseWheel` | Zoom in/out |
