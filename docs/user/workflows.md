# Workflows

## Daily Editing

- Open files from tree or quick switcher
- Keep active documents in tabs
- Use outline panel to jump headings
- Use back/forward navigation for file context switching

## Managing Blocks

- Add from selection
- Edit in popup with live preview
- Push registry state to files
- Pull file state to registry when a file version is preferred
- Use divergence indicators to prioritize updates

## Managing Prompts

- Create from selection or add button
- Organize by category
- Copy with one click

## JSON and YAML Maintenance

- Open `.json`, `.yaml`, `.yml` directly
- Use format actions for validation + pretty print

## Transcript Review

- Open `.jsonl` files
- Filter by role
- Narrow with text filter
- Expand entries to inspect full JSON

## Project Maintenance

- Use context menu in tree for create/rename/move/delete/duplicate
- Rescan after external repository changes
